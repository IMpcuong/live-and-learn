1. Comparision between DOS and Linux/UNIX:

Link: [How to create Markdown Table](https://markdown.land/markdown-table)

| DOS | Linux/UNIX |
|:---:|:----------:|
| cd | cd |
| attrib | chmod |
| comp | diff |
| copy | cp |
| del | rm |
| rd | rmdir |
| dir | ls |
| set | printenv |
| find | grep |
| help | man |
| md | mkdir |
| move | mv |
| ren (rename) | mv |
| date, time | date |
| chkdsk | df |
| type | cat |
| type \<filename\> \| more | more |
| sort | sort |

- Example:

```cmd
dir | sort /R
```
