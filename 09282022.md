1. `xargs -I` command option:

- Definition: `-I <replace-str>` ~ Replace occurrences of \<replace-str\> in the initial-arguments with names read from standard input.

- Exp:

```bash
ls output/*Response | \
  sort -t_ --key=2 -g | \
  xargs -I {} sh -c 'head -1 {}; tail {}' | \
  less

ls output/*Response | \
  sort -t_ --key=2 -g | \
  ((head -n 1) && (tail -n 10)) | \
  less
```
